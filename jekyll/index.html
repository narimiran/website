Efficient and expressive programming.

Nim is a statically typed, compiled, garbage-collected systems programming language with a design that focuses on efficiency, expressiveness, and elegance, providing unparalleled performance in an elegant package.

Nim is ...

    Efficient

  • Native code generation, not dependent on a virtual machine: Nim produces small executables without dependencies for easy redistribution.
  • The Nim Compiler runs on Windows, Linux, BSD and Mac OS X. Porting to other platforms is easy.
  • Compiles to C, C++ or JavaScript.
  • Offers hot code reloading.
  • A fast non-tracing garbage collector that supports soft real-time systems (like games).
  • Ability to manage your own memory and access the hardware directly.
  • Pointers to garbage collected memory are distinguished from pointers to manually managed memory.
  • Zero-overhead iterators.
  • Cross-module inlining.
  • Compile time evaluation of user-defined functions.
  • Whole program dead code elimination: Only used functions are included in the executable.
  • Value-based datatypes: For instance, objects and arrays are allocated on the stack.

    Expressive

  • The Nim compiler and all of the standard library are implemented in Nim.
  • Built-in high level datatypes: strings, sets, sequences, etc.
  • Modern type system with local type inference, tuples, variants, generics, etc.
  • User-defineable operators; code with new operators is often easier to read than code which overloads built-in operators.

    Elegant

  • Powerful AST-based hygienic macro system.
  • Macros can use the imperative paradigm to construct parse trees.
  • Nim does not require a different coding style for meta programming.
  • Macros cannot change Nim's syntax because there is no need for it. Nim's syntax is flexible enough.
  • Statements are grouped by indentation but can span multiple lines.
  • Indentation must not contain tabulators so the compiler always sees the code the same way as you do.

Karax

SPA framework

Karax is a framework for developing single page applications in Nim. Leverage Nim's macro system to produce a framework that allows for the development of applications that are boilerplate free.

Nimbus

Ethereum client

Nimbus aims to be a sharding client for Ethereum 2.0 Blockchain. Ethereum will potentially be on embedded systems, Nimbus will be designed to perform well on IoT and personal mobile devices.
